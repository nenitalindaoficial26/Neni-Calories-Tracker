# 🌸 Neni Calories Tracker Widget

Un widget estético y minimalista (estilo Glassmorphism flotante) para rastrear tus calorías diarias. Diseñado especialmente para integrarse en Notion. Parte de la colección exclusiva **NeniFlow™**.

## ✨ Características

- **Diseño "Frasco" Flotante:** Efecto cristal (backdrop-filter) que se funde perfectamente con el fondo de tu página en Notion.
- **5 Temas Estéticos:** Cambia con un solo clic entre Pink, Matcha 🍵, Lavender 💜, Mocha ☕ y Ice ❄️.
- **Tipografía Elegante:** Opción de alternar los números principales a una fuente fina tipo Serif.
- **Micro-Animaciones:** Anillo de progreso que brilla y se llena conforme avanzas.
- **Frases Motivadoras:** Las frases en la parte inferior evolucionan dependiendo de qué tan cerca estés de tu meta, llenándote de motivación positiva en todo el día.
- **Bilingüe:** Cambia instantáneamente la interfaz entre Español e Inglés.

## 🛠 Instalación en Notion

1. Sube los archivos (`index.html`, `style.css`, `script.js`) a un servicio de hosting como GitHub Pages, Vercel o Netlify.
2. Copia el enlace en vivo.
3. Ve a tu Notion, pega el enlace y selecciona **"Create Embed"**.
4. Ajusta las esquinas del recuadro para que el widget quede centrado y flotante a la perfección.
